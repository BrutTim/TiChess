# TiChess

Ein einfaches, konsolenbasiertes Schachspiel in Scala (MVC, funktional/immutable) mit automatisierten Tests und Coverage via scoverage.

## Entwickeln & Ausführen

### Tests

```bash
sbt test
```

### Coverage

```bash
sbt clean coverage test coverageReport
```

### Starten (Console UI)

```bash
sbt run
```
